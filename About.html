About this Weather Web

This weather web is a simple tool to help you check the current weather in any location around the world. The app provides detailed information on temperature, humidity, wind speed and more.

The data is obtained from OpenWeatherMap API and updated in real-time. The app is built using HTML, CSS, JavaScript, and Bootstrap framework.

Feel free to use this app to plan your day or to check the weather conditions in your favorite travel destinations.

Usage Guide

  1. Open the Weather Web by clicking the following URL link: weather.vercel.app.
  2. The app's home page will be displayed, and you will see a navigation bar at the top of the page with links to Home, About this Website, and User Guide.
  3. To search for weather for a city, enter the city name in the text field at the top of the page and click the Search button.
  4. The website will retrieve the weather information for the specified city and display it on the page.
  5. The page will display the name of the city whose weather is being displayed, along with the current temperature and a brief summary of the weather conditions.
  6. The page will also display three cards containing detailed weather information for the city, including temperature, humidity, and wind information.
  7. The temperature card will display the current temperature in the city, along with the minimum and maximum temperatures for the day.
  8. The humidity card will display the current humidity level in the city, along with the perceived wind and temperature levels.
  9. The wind card will display the current wind speed in the city, along with the sunrise and sunset times.
  10. The page will also display a table containing weather information for other public places.
  11. You can switch between different cities by entering the city name in the search field and clicking the Search again button.
  12. To clear the search field and start a new search, simply delete the existing text in the field and enter a new city name.
  13. You can use the navigation bar to access the User Guide or About page of this website at any time.
  14. To return to the home page, simply click the three lines in the navigation bar.
  15. Enjoy using the Weather Website to get the latest information about the weather in your city and other public places!
